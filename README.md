# Conway's Game of Life
## Cellular automaton math game

### Controls
* **Space**: start/pause game.
* **Left click**: create new cell.
* **Right click**: kill cell.

### Compile

```
$ git clone https://github.com/sriosdev/game-of-life
$ cd game-of-life
$ go build -o game-of-life
```

### Run

```
$ ./game-of-life
```
